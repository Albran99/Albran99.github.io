Epidemic Broadcast

  • C++
  • Omnetpp
  • Excel

Epidemic Broadcast is an epidemic simulation implemented using C++ in Omnetpp. The simulation is modeled as a network of nodes with peer-to-peer connections among them. Each node has a probability of sending an "infection message" to its neighbors. The goal is to find the configuration that maximizes the spread of the epidemic.
