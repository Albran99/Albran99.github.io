Epidemic Broadcast

  • C++
  • Omnetpp
  • Excel

Epidemic Broadcast is an epidemic simulation built using C++ in Omnetpp. The simulation represents a network of interconnected nodes, where each node has peer-to-peer connections. Within this network, each node has a probability of transmitting an "infection message" to its neighboring nodes. The objective of the simulation is to identify the optimal configuration that maximizes the spread of the epidemic.

Bitonic Sort

  • C++
  • Cuda
  • uProf/Nsight

BitonicSort is a study of the bitonic sort algorithm and focuses on enhancing its performance on specific hardware. The project analyzes how the utilization of multithreading and GPU accelerators can bring a significant speedup, resulting in the ability to sort an array of 2Gb in less than 15 seconds specifically on an Nvidia Tesla T4.
